Introduction

Python is an easy to learn, powerful programming language. It has efficient high-level data structures and a simple but effective approach to object-oriented programming. With its elegant syntax and dynamic typing, make it an ideal language for scripting and rapid application development in many areas on most platforms.

This tutorial introduces informally to the basic conceps and featrues of the Python language and system. It helps to have a Python interpreter handy for hands-on experience, but all examples are self-contained, so the tutorial can be read offline as well.

Whetting Your Appetite

If you do much work on computers, you find that there's some task you'd like to automate. For example, you may wish to perform a search-and-replace over a large number of files, or rename and/or rearrange a bunch of photo files. Maybe you'd like to write a small custom database, or a specialized GUI application, or a simple game. Python is the language for you.

You could write a Unix shell script to do some of these tasks, but its not well suited for GUI applications or games. You could write a program in another language, but it can take a lot of development time to get the first-draft program. Python is simpler to use, available in all operating systems, and will get the job done more quickly.

The programming language isn't named after snakes, it's named after the BBC show "Monty Python's Flying Cirucs", so references to the show and Monty Python in the documentation is the norm.

Using the Python Interpreter

To start typing code in Python, download it from the official page, and execute the Interpreter by typing in the command shell:

python3.11

The interpreter's line-editing features include interactive editing, history subsition and code completion on systmes that support the GNU Readline library. The interpreter operates similar to the Unix shell: when called with standard input connected to a tty device, it reads and executes commands interactively; when called with a file name argument or with a file as standart input, it reads and executes a script from that file.

When commands are read from a tty, the interpreter is said to be in interactive mode. In this mode it prompts for the next command with the primary prompt (>>>); for continuation lines it prompts with the secondary prompt (...).

python3.11
Python 3.11 (default, April 4 2021, 09:25:04) [GCC 10.2.0] on linux
Type "help", "copyright", "credits" or "license" for more information.
>>>
Hello world

To get started with Python and its interpreter, write after the ">>>" your first "Hello world"

>>> print("helo world")
hello world
Using Python as a Calculator

Lets continue with some Python commands. You can use the interpreter as a calculator. Use expression syntax like +,-,* and / to make mathematical operations. You can also use parentheses for grouping

>>> 2 + 2
4
>>> 50 - 5*6
20
>>> (50 - 5*6) / 4
5.0
>>> 8 / 5
1.6

The integer numbers like 4, 2, 20 have type int, the ones with a fractional part like 5.0, 1.6, have type float.

To create a variable, use the equal sign (=), no result is displayed before the next interactive prompt.

There are also other operators to make more complex equations:

  • (//) : Floor division discards the fractional part
  • (%) : Returns the remainder of the division
  • (**) : Calculate powers
>>> width = 20
>>> height = 5 * 9
>>> width * height
900
Strings

You can also manipulate strings in Python, they can be enclosed in single quotes ('...') or double quotes ("..."), all with the same result.

>>> 'spam eggs' # single quotes
'spam eggs'
>>> 'doesn\'t' # use \' to escape the single quote...
"doesn't"
>>> "doesn't" # ...or use double quotes instead
"doesn't"
>>> '"Yes," they said.'
'"Yes," they said.'
>>> "\"Yes,\" they said."
'"Yes," they said.'
>>> '"Isn\'t," they said.'
'"Isn\'t," they said.'

In here you can print the strings using the previous mentioned print() function.

>>> '"Isn\'t," they said.'
'"Isn\'t," they said.'
>>> print('"Isn\'t," they said.')
"Isn't," they said.
While Loops

We can use Python for more complicated tasks than adding two and two. For example, we can write an sub-sequence of the Fibonacci series (the sume of two elements defines the next) as follows:

>>> a, b = 0, 1
>>> while a < 10:
... (a)
... a, b = b, a+b

This example introduces several new features.

  • The first line contanins a multiple assignment: the variables a and b simultaneously get the new values 0 and 1. On the last line it is used again, demostrating that the expression on the right-hand side are all evaluated first before any of the assignments take place. The right-hand side expressions are evaluated from the left to the right.
  • The while loop executes as long as the condition (there: a < 10) remains true. In Python, any non-zero integer value is true: zero is false. In fact, any sequence; anything with a non-zero lenght is true, empty sequences are false. The comparasion operators are:
    • < (less than)
    • > (greater than)
    • == (equalt to)
    • <= (less than or equal to)
    • >= (greater than or equal to)
    • != (not equal to)
  • The body of the loop is indented. Indendation is Python's way of grouping statements. At the interactive prompt, you have to type a tab or space(s) for each indented line. In a decent text editor it will have an auto-indent facility. Note that each line within a basic block must be indented by the same amount.
Control Flow Tools

Python uses the usual flow control statements known from other languages, the most well-known statement type is the if statement.

>>> x = int(input("Please enter an integer: "))
Please enter an integer: 42
>>> if x < 0:
... x = 0
... print('Negative changed to zero')
... elif x == 0:
... print('Zero')
... elif x == 1:
... print('Single')
... else:
... print('More')
More

The for statement in Python works differently from other programming languages, Python's for statement iterates over the items of any sequence (a list of a string), in the order that they appear in the sequence.

>>> words = ['cat', 'window', 'defenestrate']
>>> for w in words:
... print(w)
...
cat
window
defenestrate

If you do need to iterate over a sequence of numbers, the function range() comes in handy, it generates arithmetic progressions:

>>> for i in range(5):
... print(i)
...
0
1
2
3
4
